Privy Token Helper

To get your Privy token:

  1. Go to your app: http://localhost:8080
  2. Make sure you're logged in
  3. Look for the "Dev Token Helper" box in the bottom-right corner
  4. Click "Get Privy Access Token"
  5. Check the browser console (F12 â†’ Console tab)

What you're seeing:

The token you found (starting with eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9) is a Supabase service token, not a Privy token.

A Privy token should:

  • Come from the usePrivy() hook's getAccessToken() method
  • Be generated when you log in via Privy (email/social/wallet)
  • Be different from Supabase tokens

Quick Debug:

Run this in your app's console to check Privy status:

// Check if Privy is loaded
console.log('Privy available?', typeof window.Privy !== 'undefined');

// Check localStorage for Privy data
const privyKeys = Object.keys(localStorage).filter(k => k.includes('privy'));
console.log('Privy localStorage keys:', privyKeys);

// Check sessionStorage too
const privySessionKeys = Object.keys(sessionStorage).filter(k => k.includes('privy'));
console.log('Privy sessionStorage keys:', privySessionKeys);
